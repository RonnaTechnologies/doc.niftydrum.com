## Description

NiftyDrum is a trigger to MIDI module.
What that means is that you can plug up to 9 piezos and 1 FSR into the 10 dedicated terminal blocks, and get MIDI messages over a USB-C connection.

## How it works

- Connect up to 9 piezos and 1 FSR to the terminal blocks
- Plug into your DAW or drum module via USB
- Use our GUI for trigger parameters, mapping and velocity shaping

## Specifications

### Hardware

- 9 Piezo inputs
- 1 FSR input (hi-hat pedal)
- Sensor connector type: terminal block
- USB type C
- Dimensions: 65x56.5mm

### Performance

- Latency: <2.5ms
- Sample rate: >10kHz
- Velocity sensitivity: 127 levels

### Software

- Cross platform: Windows, Mac, Linux
- User interface: Web-based
- Firmware updates via USB
- MIDI messages: notes, CC
